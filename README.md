# Sejoura – AI-Powered Guest Support Assistant

Sejoura is an AI-powered guest support assistant designed for homestay owners and eco-tourism operators. It helps automate customer inquiries, provide instant responses, and reduce support workload through AI-driven assistance.

## Tech Stack

- Frontend: Next.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: Supabase (PostgreSQL)
- AI: Gemini API
- Deployment: Vercel
